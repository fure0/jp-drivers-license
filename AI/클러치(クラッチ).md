# 클러치(クラッチ)란?

클러치는  
**엔진의 힘을 바퀴에 전달하거나 잠깐 끊어주는 장치**이다.

주로:

- MT(수동변속기) 자동차
- MT 오토바이

에서 사용한다.

AT(자동변속기)는 내부에서 자동 처리하기 때문에 보통 클러치 패달이 없다.

---

# 클러치의 역할

엔진은 시동이 걸리면 계속 회전하고 있다.

하지만:

- 기어를 바꿀 때
- 정차할 때
- 출발할 때

엔진과 바퀴를 잠깐 분리할 필요가 있다.

이때 사용하는 것이 클러치이다.

```text
엔진 ↔ 클러치 ↔ 바퀴
```

---

# MT 자동차의 패달 위치

```text
[클러치] [브레이크] [액셀]
   왼발      오른발    오른발
```

---

# 클러치를 밟으면

```text
엔진 ↔ 바퀴 연결 끊김
```

가능한 것:

- 기어 변경
- 정차 중 엔진 유지
- 부드러운 출발

---

# 클러치를 떼면

```text
엔진 ↔ 바퀴 연결
```

엔진의 힘이 바퀴로 전달되어 차가 움직인다.

---

# 왜 필요한가?

## 1. 기어 변경

```text
클러치 밟기
→ 기어 변경
→ 클러치 천천히 떼기
```

기어를 부드럽게 변경하기 위해 필요하다.

---

## 2. 정차

MT차는 기어가 연결된 상태에서 완전히 멈추면:

```text
엔진이 꺼질 수 있음
(エンスト)
```

그래서 정차 직전에 클러치를 밟는다.

---

## 3. 출발

클러치를 갑자기 떼면:

```text
차가 덜컹거림
또는 시동 꺼짐
```

그래서 천천히 연결해야 한다.

---

# 반클러치(半クラッチ)

클러치를 완전히 떼지 않고  
중간 정도만 연결한 상태.

```text
완전 분리 ❌
완전 연결 ❌
중간 연결 ⭕
```

주로 사용되는 상황:

- 출발
- 언덕길 출발
- 저속 이동
- 주차

---

# 오토바이의 클러치

보통:

```text
왼손 = 클러치 레버
왼발 = 기어 변경
```

사용 순서:

```text
클러치 잡기
→ 기어 변경
→ 천천히 놓기
```

---

# AT 자동차와 차이

AT는 클러치 조작을 자동으로 처리한다.

그래서 운전자는:

- 브레이크
- 액셀

만 사용하면 된다.

---

# 일본어 표현

- クラッチ = 클러치
- クラッチペダル = 클러치 패달
- 半クラッチ = 반클러치
- 半クラ = 반클
- エンスト = 시동 꺼짐
- ギアチェンジ = 기어 변경
